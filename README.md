## 👩‍💻 À propos

Je suis ingénieure en systèmes embarqués passionnée par le **firmware bas niveau**, les **protocoles de communication** et l'**intégration matériel-logiciel**. Je construis des systèmes qui tournent là où les ressources sont rares et la fiabilité est critique.

Actuellement en stage DevOps Embedded chez **Unistellar** (Marseille), je travaille sur le firmware BLE et les pipelines CI/CD du produit **Envision** — une lunette connectée STM32.

Avant ça : échange académique à l'**ENSEA Cergy** (S8, IoT/Électronique), et un stage chez **SAGEMCOM** où j'ai développé une interface Python pour tester les protocoles de communication de compteurs intelligents.

> *"I write code that runs on hardware with no OS, no second chances, and 256KB of RAM."*
